# GoMed_Redo- A Medical ChatBot Project 

## Overview

Welcome to the Medical ChatBot project! This project aims to provide a user-friendly interface for interacting with a medical chatbot powered by advanced natural language processing. The chatbot is designed to assist users with medical-related queries, provide information on symptoms, suggest potential conditions, and offer general healthcare advice.

## Getting Started

### Prerequisites

- Docker installed on your system: [Docker Installation Guide](https://docs.docker.com/get-docker/)

### Setup

1. Clone the repository:

    ```bash
    git clone https://github.com/your-username/medical-chatbot.git
    cd medical-chatbot
    ```

2. Build the Docker image:

    ```bash
    docker build -t medical-chatbot:latest .
    ```

3. Run the Docker container:

    ```bash
    docker run -p 5000:5000 medical-chatbot:latest
    ```

4. Access the Medical ChatBot at [http://localhost:5000](http://localhost:5000) in your web browser.

## Usage

1. Open the web browser and navigate to [http://localhost:5000](http://localhost:5000).
2. Enter your medical-related queries in the chat interface.
3. The chatbot will process your input and provide relevant information and suggestions.

## Features

- **Natural Language Processing (NLP):** The chatbot utilizes advanced NLP techniques to understand and respond to user queries effectively.

- **Medical Database:** A comprehensive medical database is integrated to provide accurate information about symptoms, conditions, and general healthcare advice.

- **User-Friendly Interface:** The web-based interface makes it easy for users to interact with the chatbot without any technical expertise.
